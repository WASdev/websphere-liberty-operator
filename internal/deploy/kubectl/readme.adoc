
== Installing WebSphere Liberty Operator with kubectl

The WebSphere Liberty Operator can be installed using Kubernetes command-line interface (CLI) (kubectl) to:

* watch own namespace
* watch another namespace
* watch all namespaces in the cluster

Follow the instructions in the link:++https://www.ibm.com/docs/en/was-liberty/base?topic=operator-installing-kubernetes-cli#in-t-kubectl__non-olm++[documentation], but replace `https://raw.githubusercontent.com/WASdev/websphere-liberty-operator/main/deploy/releases/1.5.1/kubectl/` with `https://raw.githubusercontent.com/WASdev/websphere-liberty-operator/deploy-1.5.2/internal/deploy/kubectl/` in the commands.
